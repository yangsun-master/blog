How to create a blog

I plan to have a personal blog. Here’s what I want:

  • My blog is a technique blog, it will contain code. The blog should display code nicely.
  • I want source control, so I can easily know what change I made.
  • I want to highly customize to my blog, and no ads.
  • I want my personal domain.
  • I want the blog to be a sub folder of my domain.

This is my solution:
Markdown + Hexo + Github pages

Let’s start!

Get a Github account.

Install git

Install Nodejs

Create two repos.

One for blog source code, the other for the blog website. https://help.github.com/en/articles/create-a-repo

On your local box, clone your repo.

Install hexo

Initialize the blog source code folder by hexo.

Run npm to install packages.

Create your first post.

The file hello-world.md will be created under _posts folder.

Generate the static files.

Publish to the blog website repo.

Copy all the files under public folder to the blog web site repo and checkin.

Enable Github pages.

Click on the Settings tab and scroll down to the GitHub Pages section.

The Github pages should be ready to go.
http://YOUR_ACCOUNT.github.io/blog_website

Custom domain

Create a CNAME in your DNS provider, config www mapping to YOUR_ACCOUNT.github.io
Config the your domain in the Github pages
Check “Enforce HTTPS”.

You’re all done.
